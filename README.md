# Juego de ahorcado
Tarea del juego de ahorcado

Descripción: 
Este proyecto consiste en un juego de ahorcado desarrollado en Python.
El programa selecciona una palabra aleatoria de la lista y el usuario debe adivinar letra por letra hasta completar la palabra.

Instalación:
1. Tener python instalado en la computadora.
2. Descarggar el archivo #Tarea Juego de ahorcado.py
3. Abrir el archivo en cualquier entorno de desarrollo compatible.
4. Ejecutar el programa

Usabilidad:
1. Ejecutar el programa.
2. El sistema seleccionará una palabra aleatoria.
3. La palabra aparecerá oculta en guiones bajos.
4. El usuario deberá ingresar letras desde el teclado.
5. Cada letra correcta se mostrará en pantalla.
6. El juego finaliza cuando la palabra ha sido descubierta.
